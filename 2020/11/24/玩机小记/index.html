HF Patch

1.for AI-Shoujo

先找到Releases · ManlyMarco/AI-HF_Patch · GitHub

发布页地址1：HF Patch

发布页地址2：HF Patch for AI-Shoujo v2.x

找到任意磁力链接下载即可

注意事项1：建议将Patch安装到Game catalog

注意事项2：安装过程中，会提示你选择要安装的组件，默认

会帮你勾选English translation,要取消勾选，以免把自身

的汉化给覆盖了

注意事项3：安装后，打开Game catalog，你会发现有文件

夹KKManager，此时需要更新至最新版，更新方式如下：首先

进入发布页地址：KKManager v0.XX.0，找到最新版点击下载。

下载完成后，解压覆盖原文件即可完成更新。
