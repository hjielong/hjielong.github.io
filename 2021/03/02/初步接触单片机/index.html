单片机基础初步整理

备注：以下讨论主要围绕80C51/89C51，注意一下内容还处于初步整理阶段

组成

2个电源引脚:(20/40)

Vcc Vss(Gnd)
电源端，接+5V电源 接地端，接+5V电源地端

2个时钟引脚:(18/19)

XTAL1 XTAL2
片内振荡电路反相放大器输入 片内振荡电路反相放大器输出

4个控制引脚:(9/29/30/31)
RST (9脚) 复位信号输入端，高电平有效。
PSEN (29脚) 外部程序存储器读选通输出信号
ALE/PROG (30脚) 地址锁存允许输出端/编程脉冲输入端。
EA/Vpp (31脚) 外部程序存储器地址使能输入/编程电压输入端。
全称 说明 注意事项
CPU Central Processing Unit 由运算和控制逻辑组成，同时还包括中断系统和部分外部特殊功能寄存器
RAM 随机存储器 Random-Access Memory 用以存放可以读写的数据，如运算的中间结果、最终结果以及欲显示的数据 掉电丢失
ROM 只读存储器 Read-Only Memory 用以存放程序、一些原始数据和表格 掉电不丢失
I/O input/output 四个8位并行I/O口，既可用作输入，也可用作输出
T/C timer/counter 两个定时/记数器，既可以工作在定时模式，也可以工作在记数模式

电平

数字电路中的电平

高电平 低电平
3.3v或5v 0v

RS232电平(计算机串口电平)

高电平 低电平
+12v -12v

数据类型

类型 关键字 所占位数
整型 unsigned int 16
整型 unsigned short int 16
整型 unsigned long int 32
字符型 unsigned char 8

在C中，默认的基础数据类型均为signed，如定义变量为int，long等，都为有符号的。

如果要定义无符号类型，必须显式地在变量类型前加unsigned。

基础程序与语句

大致和c相似，这里补充一点

define LED P0

这里用LED来指代P0，相当于给P0端口起名字，注意语句后不接分号。

typedef unsigned int uint;

这语句常用来重新定义关键字，这里就是用uint来指代unsigned int。
