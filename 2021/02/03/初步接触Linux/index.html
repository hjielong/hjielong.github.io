    1. 2.1. Shell
    2. 2.2. 命令初探
      1. 2.2.1. &emsp;&emsp;~ man：查看命令帮助信息
      2. 2.2.2. &emsp;&emsp;~ - -help：查看命令帮助信息(精简)
      3. 2.2.3. &emsp;&emsp;~ clear：清除屏幕
      4. 2.2.4. &emsp;&emsp;~ who / w：打印当前登录用户
      5. 2.2.5. &emsp;&emsp;~ date：显示日期与时间
      6. 2.2.6. &emsp;&emsp;~ cal：显示日历
      7. 2.2.7. &emsp;&emsp;~ whatis + 命令：显示该命令是什么
      8. 2.2.8. &emsp;&emsp;~ whereis + 命令：显示该命令路径
      9. 2.2.9. &emsp;&emsp;~ shutdown：关闭服务器
      10. 2.2.10. &emsp;&emsp;~ logout：退出当前用户登录
      11. 2.2.11. &emsp;&emsp;~ history：显示与操纵历史命令(-c 清空当前历史命令列表)
    3. 2.3. 快捷键
    4. 2.4. 文件管理
      1. 2.4.1. &emsp;&emsp;~ pwd：print working directory 显示当前工作目录
      2. 2.4.2. &emsp;&emsp;~ cd：change directory 切换工作目录
      3. 2.4.3. &emsp;&emsp;~ ls：list directory contents 列出当前目录所含文件及子目录
      4. 2.4.4. &emsp;&emsp;~ cat：concatenate 连接文件并将内容打印到标准输出设备上，可用来查看文件
      5. 2.4.5. &emsp;&emsp;~ mkdir：make directory 创建文件夹
      6. 2.4.6. &emsp;&emsp;~ rmdir：remove empty directory 删除空目录
      7. 2.4.7. &emsp;&emsp;~ touch：修改文件或者目录的时间属性，包括存取时间和更改时间。若文件不存在，系统会建立一个新的文件
      8. 2.4.8. &emsp;&emsp;~ rm：remove 移除文件或目录
      9. 2.4.9. &emsp;&emsp;~ cp：copy 复制文件或目录
      10. 2.4.10. &emsp;&emsp;~ mv：move 移动文件
      11. 2.4.11. &emsp;&emsp;~ file: 识别文件类型，也可用来辨别一些文件的编码格式
      12. 2.4.12. &emsp;&emsp;~ wc: word count 可以计算文件的Byte数、字数、或是列数
      13. 2.4.13. &emsp;&emsp;~ shell的echo指令： 用于字符串的输出
      14. 2.4.14. &emsp;&emsp;~ sort: 将文本文件内容加以排序
      15. 2.4.15. &emsp;&emsp;~ uniq: 检查及删除文本文件中重复出现的行列
      16. 2.4.16. &emsp;&emsp;~ iconv：转换文件的编码方式
      17. 2.4.17. &emsp;&emsp;~ grep：查找文件里符合条件的字符串
    5. 2.5. 网络相关
      1. 2.5.1. &emsp;&emsp;~ wget：从指定的URL下载文件
