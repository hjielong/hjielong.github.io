初步接触Linux

发表于2021-02-03|更新于2021-02-16|个人笔记
|阅读量:

写在前面

编码：

  ~ gb2312：国标码（常用汉字、平假名、片假名）

  ~ gbk：国标码扩展码

  ~ gb18030：汉字、少数民族文字

字符集：万国码

  ~ Unicode：实现（utf-8\utf-16\utf-32）

硬件(hardware)：运算器、控制器、存储器(RAM)、输入设备、输出设备

软件(software)：系统软件和应用软件

操作系统(Operating System)：

  ~ Windows

  ~ Macintosh(macOS)

  ~ Unix

  ~ Linux

编程语言：常见语言排行

  ~ Julia / Python / R

云服务器：

  ~ 阿里云

  ~ 发布页

  ~ 连接方式：SSH协议(安全的远程连接)

  ~ SSH客户端工具： FinalShell

Linux

Shell

bash(Bourne Again Shell) ~ 壳程序 ~ 人和计算机交互的界面

命令初探

在线查询：常用命令手册

  ~ 一般表达格式：命令 + 参数 + 命令作用对象

  ~ man：查看命令帮助信息

  ~ - -help：查看命令帮助信息(精简)

  ~ clear：清除屏幕

  ~ who / w：打印当前登录用户

  ~ date：显示日期与时间

  ~ cal：显示日历

  ~ whatis + 命令：显示该命令是什么

  ~ whereis + 命令：显示该命令路径

  ~ shutdown：关闭服务器

  ~ logout：退出当前用户登录

  ~ history：显示与操纵历史命令(-c 清空当前历史命令列表)

留有历史命令的好处：

  按上下键可以查看之前命令；

  ！+数字 可以执行某指定命令；

  ！！执行刚刚的命令。

快捷键

  ~ Tab：命令补全和路径补全

 如果敲击一次Tab键未能补全，再敲多一次会自动显示多个可能补全的命令

  ~ ctrl + c：终止一个命令的执行

  ~ ctrl + w：删除一个单词

  ~ ctrl + u：删除整行

  ~ ctrl + a：光标回到行首

  ~ ctrl + e：光标回到行尾

  ~ ctrl + d：结束输入

  ~ ctrl + z：把一个正在运行的命令暂停放到后台

文件管理

  ~ pwd：print working directory 显示当前工作目录

  ~ cd：change directory 切换工作目录

  ~ ls：list directory contents 列出当前目录所含文件及子目录

  ~ cat：concatenate 连接文件并将内容打印到标准输出设备上，可用来查看文件

  ~ mkdir：make directory 创建文件夹

  ~ rmdir：remove empty directory 删除空目录

  ~ touch：修改文件或者目录的时间属性，包括存取时间和更改时间。若文件不存在，系统会建立一个新的文件

  ~ rm：remove 移除文件或目录

  ~ cp：copy 复制文件或目录

  ~ mv：move 移动文件

  ~ file: 识别文件类型，也可用来辨别一些文件的编码格式

  ~ wc: word count 可以计算文件的Byte数、字数、或是列数

  ~ shell的echo指令： 用于字符串的输出

  ~ sort: 将文本文件内容加以排序

  ~ uniq: 检查及删除文本文件中重复出现的行列

  ~ iconv：转换文件的编码方式

  ~ grep：查找文件里符合条件的字符串

网络相关

  ~ wget：从指定的URL下载文件
