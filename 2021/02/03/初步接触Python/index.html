  ~ Unix

  ~ Linux

编程语言：常见语言排行

  ~ Julia / Python / R

云服务器：

  ~ 阿里云

  ~ 发布页

  ~ 连接方式：SSH协议(安全的远程连接)

  ~ SSH客户端工具： FinalShell

Linux

Shell

bash(Bourne Again Shell) ~ 壳程序 ~ 人和计算机交互的界面

命令初探

在线查询：常用命令手册

  ~ 一般表达格式：命令 + 参数 + 命令作用对象

  ~ man：查看命令帮助信息

  ~ - -help：查看命令帮助信息(精简)

  ~ clear：清除屏幕

  ~ who / w：打印当前登录用户

  ~ date：显示日期与时间

  ~ cal：显示日历(-3 参数 显示最近3个月的日历)

  ~ whatis + 命令：显示该命令是什么

  ~ whereis + 命令：显示该命令路径

  ~ shutdown：关闭服务器

  ~ logout：退出当前用户登录

  ~ history：显示与操纵历史命令(-c 清空当前历史命令列表)

留有历史命令的好处：

  按上下键可以查看之前命令；

  ！+数字 可以执行某指定命令；

  ！！执行刚刚的命令。

快捷键

  ~ Tab：命令补全和路径补全

 如果敲击一次Tab键未能补全，再敲多一次会自动显示多个可能补全的命令

  ~ ctrl + c：终止一个命令的执行

  ~ ctrl + w：删除一个单词

  ~ ctrl + u：删除整行

  ~ ctrl + a：光标回到行首

  ~ ctrl + e：光标回到行尾

  ~ ctrl + d：结束输入

  ~ ctrl + z：把一个正在运行的命令暂停放到后台

文件管理

  ~ pwd：print working directory 显示当前工作目录

  ~ cd：change directory 切换工作目录

  ~ ls：list directory contents 列出文件夹的内容

  ~ cat：concatenate 查看文件内容

  ~ mkdir：make directory 创建文件夹

  ~ rmdir：remove empty directory 删除空目录

  ~ touch：创建空文件或者修改文件的最后访问时间

  ~ rm：remove 移除文件或目录

  ~ ctrl + c：终止一个命令的执行

  ~ ctrl + c：终止一个命令的执行

  ~ ctrl + c：终止一个命令的执行
