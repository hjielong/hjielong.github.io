准备工作

新建一个H-GIE_1.x版本文件夹

进入文件夹后，在当前目录上方路径显示框中输入cmd，回车进入cmd

全局升级模块

升级模块hexo-cli:输入命令npm install hexo-cli -g

新建博客

输入命令hexo init创建博客

全局安装模块

安装模块npm-check:输入命令npm install -g npm-check

更新

更新前，简单了解npm-check的用法

开始更新

输入命令npm-check -u,在当前文件夹进行交互式更新

查看hexo版本信息

输入命令hexo -v，检查更新正常后准备工作到此结束。
